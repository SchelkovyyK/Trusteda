# ЁЯУК TrustEDA

TrustEDA is a full-stack web platform for exploratory data analysis (EDA), automatic statistical testing, and AI-assisted interpretation of results.

The system allows users to upload CSV datasets, generate intelligent analysis suggestions, run statistical tests between columns, evaluate reliability and stability of findings, and obtain natural-language explanations generated by an LLM.

---

# ЁЯЪА Features

## Dataset Management

- Upload CSV datasets
- Automatic dataset preview
- Dataset metadata
- Dataset history
- Delete datasets
- Local storage without external database

---

## Intelligent Suggestion Engine

Automatically generates analysis candidates based on column types.

Supported suggestions:

| Column Types        | Analysis             |
| ------------------- | -------------------- |
| Numeric + Numeric   | Correlation          |
| Category + Numeric  | Comparison           |
| Category + Category | Association Analysis |
| Date + Numeric      | Trend Analysis       |

Suggestions are ranked by priority:

- High priority
- Medium priority
- Low priority

Suggestion history is stored per dataset.

---

## Statistical Analysis Engine

The platform automatically selects the appropriate statistical method.

### Correlation Analysis

Used for:

Numeric + Numeric

Methods:

- Pearson correlation

Metrics:

- Correlation coefficient
- p-value

---

### Comparison Analysis

Used for:

Category + Numeric

Methods:

- ANOVA

Metrics:

- F-statistic
- p-value

---

### Association Analysis

Used for:

Category + Category

Methods:

- Chi-Square test

Metrics:

- Chi-square statistic
- p-value
- Cramer's V

---

### Trend Analysis

Used for:

Date + Numeric

Methods:

- Correlation on temporal sequence

Metrics:

- Correlation coefficient
- p-value

# ЁЯУИ Reliability System

TrustEDA evaluates not only significance but also trustworthiness.

---

## Stability

Bootstrap resampling is used.

Repeated analysis is performed on multiple random samples.

Metric:

```
0 тЙд stability тЙд 1
```

High values indicate that results remain similar across samples.

---

## Reliability

Combined metric:

```
reliability =
0.4 ├Ч significance +
0.3 ├Ч effect size +
0.3 ├Ч stability
```

Range:

```
0 тЙд reliability тЙд 1
```

Interpretation:

| Reliability | Meaning       |
| ----------- | ------------- |
| >0.9        | Very reliable |
| 0.7тАУ0.9     | Reliable      |
| 0.5тАУ0.7     | Moderate      |
| <0.5        | Weak          |

---

# ЁЯУЪ Metric Explanations

## p-value

Measures statistical significance.

Interpretation:

| p-value | Meaning         |
| ------- | --------------- |
| <0.05   | Significant     |
| >0.05   | Not significant |

---

## Correlation Coefficient

Measures strength of linear relationship.

Range:

```
-1 to 1
```

Interpretation:

| Value | Meaning         |
| ----- | --------------- |
| 0     | No relationship |
| ┬▒0.3  | Weak            |
| ┬▒0.5  | Moderate        |
| ┬▒0.7  | Strong          |
| ┬▒1    | Perfect         |

---

## Chi-square Statistic

Measures deviation between observed and expected frequencies.

Higher values indicate stronger differences.

---

## Cramer's V

Measures association strength between categorical variables.

Range:

```
0 to 1
```

Interpretation:

| Value   | Meaning  |
| ------- | -------- |
| <0.1    | Weak     |
| 0.1тАУ0.3 | Moderate |
| >0.3    | Strong   |

---

## F-statistic

Used in ANOVA.

Larger values indicate greater differences between groups.

---

## Stability

Measures reproducibility of results.

Range:

```
0 to 1
```

Higher values indicate consistent findings.

---

## Reliability

Overall confidence score.

Range:

```
0 to 1
```

Combines:

- statistical significance;
- effect size;
- stability.

# API Endpoints

## Upload Dataset

```http
POST /upload
```

Uploads CSV dataset.

---

## List Files

```http
GET /files
```

Returns all datasets.

---

## Dataset Details

```http
GET /dataset/{file_id}
```

Returns:

- metadata;
- preview;
- reports;
- suggestions.

---

## Analyze Columns

```http
POST /analyze
```

Parameters:

- col1
- col2

- analysis metrics;
- stability;
- reliability;
